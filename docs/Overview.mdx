# Welcome!

#### The boilerplate is a `developer-friendly` highly customizable template that makes it easy to build your own dream application.

Customize the project with your own theme. [MUI](https://mui.com) makes it easy to change the colors, typography, and other aspects of your project's style. This makes it easy to keep track of your project's styles and to make changes to them as needed.

## Introduction

Hi there! We're glad you're using our theme. We hope you find it helpful. The documentation are built with [Material UI](https://mui.com). Our documentation is built on top of Material UI, a popular [React](https://react.dev) UI library that provides a consistent and accessible design system. This means that our docs look great and are easy to use, regardless of the device or browser you are using.
