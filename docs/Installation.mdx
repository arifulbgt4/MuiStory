# Installation

**To install the project, open the project folder in a terminal and follow the instruction.**

> If you run into any issues installing the project, please don't hesitate to reach out to me. email me at: arifulbgt4@gmail.com

## Requirements

- [Node.js](https://nodejs.org/en) 18.x.
- Do not delete the `package-lock.json` / `yarn.lock` file in the project.

## Package Manager

**Recommanded to use the [yarn](https://yarnpkg.com) package manager**

- [NPM](https://www.npmjs.com)
- [YARN](https://yarnpkg.com) **(Recommanded)**

> There might be other package managers that were not listed here.

## Install dependencies

Open the project folder in a terminal and install all dependencies

```ts
yarn install
```

or

```ts
npm i
```

or

```ts
npm i --legacy-peer-deps
```

## Start - Next.JS development server

Once the installation is complete, you are now ready to run your App:

```ts
yarn dev
```

or

```ts
npm run dev
```

## Start - Storybook

Once the installation is complete, you are now ready to run your Storybook:

```ts
yarn storybook
```

or

```ts
npm run storybook
```

> **Please note that this project is a front-end only, and does not handle any backend logic or databases. This means that you will need to provide your own backend, or use an existing one.**
