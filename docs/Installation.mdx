# Installation

## Package Manager

Recommande to use the [yarn](https://yarnpkg.com) package manager

- [NPM](https://www.npmjs.com)
- [YARN](https://yarnpkg.com) <b>(Recommande)</b>

## Requirements

- [Node.js](https://nodejs.org/en) 16.x.
- Do not delete the `package-lock.json` / `yarn.lock` file in the project.

## Install dependencies

Open the project folder and install all dependencies

```ts
yarn install
```

or

```ts
npm i
```

or

```ts
npm i --legacy-peer-deps
```

## Start - Next.JS development server

Once the installation is done, Now ready to run your app:

```ts
yarn dev
```

or

```ts
npm run dev
```

## Start - Storybook

Once the installation is done, Now ready to run your storybook:

```ts
yarn storybook
```

or

```ts
npm run storybook
```
